# Portfolio Case Study: WPM Trainer

## Problem
Typing tests usually measure speed once and leave the user guessing what to practice next. A better trainer should surface mistakes and turn them into focused drills.

## Build
WPM Trainer is a zero-dependency browser app with timed tests, live WPM and accuracy, per-character error highlighting, weak-key tracking, custom text, progress history, and a finger-position keyboard.

## Why it is useful
- Helps beginners practice the exact keys they miss.
- Saves progress locally with no account or backend.
- Works as a small portfolio app that demonstrates state, UI feedback, localStorage, and polish.
- Fits a tiny public demo for job hunting or a learning-tool bundle.
